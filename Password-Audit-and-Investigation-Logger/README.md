# Password Audit and Investigation Logger

## Overview

This Python application simulates a password auditing tool by collecting password-related information, evaluating password strength, and recording investigation results.

## Features

- Collects password information
- Evaluates password strength
- Records audit results
- Demonstrates logging and user input validation

## Skills Demonstrated

- Python
- User Input Validation
- Functions
- Logging
- Exception Handling
- Password Security Fundamentals
