# Cybersecurity File Inventory and Integrity Report

## Overview

This Python application inventories files within a specified directory and generates a report to help verify file integrity and support basic cybersecurity investigations.

## Features

- Scans files within a directory
- Generates a file inventory report
- Displays file information
- Demonstrates file handling and reporting

## Skills Demonstrated

- Python
- File I/O
- Directory Traversal
- Functions
- Exception Handling
- Cybersecurity File Management
